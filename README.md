Website ini dikembangkan sebagai bagian dari AFL 3 mata kuliah Web Programming. Website ini mencakup 6 halaman utama (Home, About, Project, Skills, Gallery, dan Contact).

Fitur Utama:
Multi-page Navigation: Navigasi antar halaman yang lancar.
Responsive Design: Tampilan yang optimal di berbagai perangkat (Desktop, Tablet, dan Mobile).
Interactive Portfolio: Galeri dan daftar projek menggunakan sistem Modal Bootstrap untuk detail yang lebih dalam.
Form Validation: Validasi formulir kontak menggunakan JavaScript.
Skill Visualization: Animasi progress bar untuk menunjukkan tingkat kemahiran teknis.

Implementasi:
HTML5 untuk struktur konten
CSS untuk styling tampilan
Bootstrap sebagai Framework layout grid dan komponen UI
Bootstrap icon untuk mengakses kumpulan ikon vektor
JS logika interaksi dan animasi
